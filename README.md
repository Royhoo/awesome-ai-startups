## 项目列表

### 1. n8n
- **GitHub 地址：** https://github.com/n8n-io/n8n
- **Stars：** ⭐ 178,000+
- **领域：** AI 工作流自动化 / No-Code
- **核心功能：** 可视化拖拽式工作流平台，原生支持 AI 节点和 LangChain 集成，400+ 服务集成，支持自部署
- **创业 / 落地价值：** 极低门槛搭建 AI 自动化服务，可为企业客户提供邮件自动分类、内容生产流水线、客服 Agent 等付费服务，fair-code 许可允许商用

---

### 2. AutoGPT
- **GitHub 地址：** https://github.com/Significant-Gravitas/AutoGPT
- **Stars：** ⭐ 182,000+
- **领域：** AI Agent / 自主任务执行
- **核心功能：** 自主规划、执行多步任务的 AI Agent 平台，支持插件扩展、网络搜索、代码执行、文件操作
- **创业 / 落地价值：** 可二次开发为垂直行业 AI 助理（法律、财务、运营），核心团队已成立公司并融资，生态活跃

---

### 3. Dify
- **GitHub 地址：** https://github.com/langgenius/dify
- **Stars：** ⭐ 132,000+
- **领域：** AI 应用开发平台 / LLMOps
- **核心功能：** 生产就绪的 AI 工作流开发平台，内置 RAG 管道、多模型支持、Agent 编排、使用监控，支持 MCP 协议
- **创业 / 落地价值：** 可快速搭建企业级 AI 产品（客服机器人、知识库问答、内部工具），官方提供云服务商业版，自部署版本免费可商用

---

### 4. Langflow
- **GitHub 地址：** https://github.com/langflow-ai/langflow
- **Stars：** ⭐ 145,000+
- **领域：** AI Agent / RAG 可视化构建
- **核心功能：** 基于 LangChain 的低代码 AI Agent 和 RAG 工作流构建平台，拖拽组装后可一键部署为 API
- **创业 / 落地价值：** 快速原型验证 AI 产品，已被 DataStax 收购并推出云服务，是 RAG 产品创业的最佳起点之一

---

### 5. Open WebUI
- **GitHub 地址：** https://github.com/open-webui/open-webui
- **Stars：** ⭐ 126,000+
- **领域：** AI 对话界面 / 私有 ChatGPT
- **核心功能：** 功能完整的自托管 AI 对话平台，支持 Ollama/OpenAI API，内置 RAG、语音通话、角色扮演、RBAC 权限管理
- **创业 / 落地价值：** 可为企业/团队部署私有 AI 平台，一行命令安装，支持 SSO 和审计日志，适合做企业 AI 工具 SaaS

---

### 6. Ollama
- **GitHub 地址：** https://github.com/ollama/ollama
- **Stars：** ⭐ 164,000+
- **领域：** 本地 LLM 部署
- **核心功能：** 轻量级本地大模型运行框架（Go 编写），支持 Llama、Mistral、DeepSeek、Gemma 等，一条命令即可运行
- **创业 / 落地价值：** 本地 AI 基础设施核心组件，适合构建数据不出境的企业 AI 解决方案，与 Open WebUI 搭配即可提供完整私有部署服务

---

### 7. Flowise

- **核心功能：** 专为 LLM 应用设计的拖拽式构建工具，深度整合 LangChain，支持 Chatflow 和 Agentflow 两种模式
- **创业 / 落地价值：** 特别适合快速交付客户定制聊天机器人，部署简单（Docker 一键启动），有官方云服务，适合接外包 AI 项目
